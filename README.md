# SnapSum4J - 数字识别与求和工具

SnapSum4J 是一款基于 OCR 技术的数字识别与求和工具，能够快速识别图片或屏幕中的数字并自动计算总和，适用于需要快速统计大量数字的场景，如发票金额、报表数据等。

## 功能特点

- 📷 **图片识别**：支持上传图片并选择区域进行数字识别
- 🖥️ **屏幕截图**：支持直接截取屏幕区域进行数字识别
- 🧮 **自动求和**：识别完成后自动计算所有数字的总和
- ✏️ **手动编辑**：支持手动编辑识别结果，实时更新总和
- 📌 **窗口置顶**：支持窗口置顶，方便边操作边查看结果
