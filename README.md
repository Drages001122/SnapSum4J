# SnapSum4J - 数字识别与求和工具

## 📚 项目简介

SnapSum4J 是一个高效、直观的数字识别与求和工具，基于 Python 开发，能够快速识别图片中的所有数字并自动计算它们的总和。该工具通过友好的可视化界面，让用户无需编程知识即可轻松完成数字识别和求和操作，适用于各种需要快速处理图片中数字的场景。

## ✨ 功能特性

- **高精度数字识别**：基于 PaddleOCR 库，支持识别图片中的所有数字（包括小数）
- **直观可视化界面**：采用 Tkinter 构建的用户友好界面，操作简单易懂
- **灵活数字管理**：支持手动添加、编辑和删除数字，满足各种场景需求
- **智能自动求和**：识别完成后自动计算总和，用户修改数字后实时更新
- **清晰结果展示**：结构化展示识别到的数字和最终求和结果
- **多格式图片支持**：兼容常见图片格式（JPG、PNG、BMP 等）
- **实时状态提示**：识别过程中显示红色提示，识别完成后显示蓝色结果和耗时

## 🖥️ 使用方法

1. **上传图片**：点击"选择图片"按钮，浏览并选择包含数字的图片
2. **自动识别**：系统会自动识别图片中的所有数字，识别过程中会显示红色提示
3. **查看结果**：识别完成后，识别到的数字会显示在文本框中（每行一个），同时会自动计算并显示总和
4. **编辑数字**：在文本框中可以手动添加、修改或删除数字（每行一个），修改后系统会自动更新总和
5. **查看状态**：系统会实时显示操作状态，包括识别耗时和计算结果
